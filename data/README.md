# Derived data

## `drumlin_centroids.csv`

41,702 drumlins from the BRITICE v2 *Subglacial lineations (polygons)* layer
(Great Britain), each reduced to a centroid with a principal-axis (flow)
azimuth, width and length, plus a coherent-field label.

| column | units | description |
|---|---|---|
| `easting_km` | km | British National Grid easting of the polygon centroid |
| `northing_km` | km | British National Grid northing of the polygon centroid |
| `flow_azimuth_deg` | degrees (0–180) | long-axis azimuth = local ice-flow direction |
| `width_m` | m | minor-axis extent (drumlin width) |
| `length_m` | m | major-axis extent (drumlin length) |
| `field_label` | integer | coherent-field id from `code/segment_fields.py` |

Generated by `code/parse_britice.py` (shapefile → centroids) followed by
`code/segment_fields.py` (field labels). Nine fields with ≥300 drumlins and
azimuth concentration ≥0.72 are used for inference in the paper.
